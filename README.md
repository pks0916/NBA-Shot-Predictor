# NBA Shot Predictor (AI 4705 Final Project)

## Overview
This project predicts whether an NBA shot will be **made (1)** or **missed (0)** using the 2025 season shot chart dataset.

Models used:
- Logistic Regression
- K-Nearest Neighbors
- Gaussian Naive Bayes
- Kernel Logistic Regression
- PCA + all 4 models
